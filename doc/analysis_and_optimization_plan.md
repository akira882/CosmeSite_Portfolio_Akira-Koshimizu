# オルビス（ORBIS）LP最適化分析レポートと実装計画

## 1. 現状分析 (Current Status Analysis)

現在のLPを「オルビスのサイト戦略」というフィルターを通して深層分析した結果、以下の乖離（ギャップ）と課題が浮き彫りになりました。

### ✅ 評価できる点 (Positives)
*   **モデル選定:** 最近の変更で、過度なメイク感が消え、オルビスらしい「素肌感」「意思のある女性」のイメージに近づいています。
*   **ターゲットインサイト:** `Problem` セクションで「オフィスの乾燥」「夕方のくすみ」に触れている点は、働く女性のリアルな悩みを捉えています。
*   **Project Insight:** 採用担当者へ向けたメタ的な視点提示は、戦略的思考のアピールとして有効です。

### ⚠️ 本質的な課題とギャップ (Critical Gaps)

#### ① 「高級感」と「ここちよさ」のミスマッチ
*   **現状:** 黒やダークトーンを多用した「夜の高級感」「外資系デパコス」のような重厚な世界観。
*   **オルビスの戦略:** 「ここちを美しく。」「日常のケの美」。
*   **分析:** オルビスのデザインコードは、**「白」「光」「水」「透明感」**です。現在のダークなデザインは、オルビスが大切にする「日常の延長にあるスキンケア」という親しみやすさや、誠実なクリーンさを阻害しています。

#### ② 「雰囲気」と「誠実さ」のバランス
*   **現状:** 英語のタイトル (`Lumière Éclat`) や装飾的なフォント (`Cinzel`) が目立ち、イメージ先行型になっている。
*   **オルビスの戦略:** 「誠実であること」「結果主義」。
*   **分析:** オルビスのLPは、**「日本語のキャッチコピー」**が主役です。機能や効果を、心地よい言葉で、しかし論理的に伝える必要があります。今のままでは「雰囲気美人のための商品」に見え、「肌悩みを科学で解決したい層」に刺さりません。

#### ③ ターゲット年齢層とのズレ
*   **現状:** 「28歳丸の内OL」というペルソナ設定が、少し若年層（初期エイジング）寄り。
*   **オルビスの戦略:** 30代～60代のエイジングケア層が主力。
*   **分析:** モデルの年齢感や悩みの描写を、もう少し**大人の余裕**や**本質的な肌質改善**へとシフトさせる必要があります。

## 2. 最適化方針 (Optimization Strategy)

「オルビスらしさ」を体現し、採用担当者に「このまま本番で使える」と思わせる品質へ引き上げるための最適化方針です。

### デザインコンセプト: **「Scientific Clarity（科学的な透明感）」**

1.  **カラーパレット刷新:**
    *   Before: `bg-brand-900` (Dark Brown) / Gold
    *   After: **`bg-white` (Clean White) / `text-slate-700` (Gray) / `accent-water-blue` (Clear Blue)**
    *   全体を「白」と「光」の空間にし、商品の信頼感とテクスチャーの心地よさを表現します。

2.  **タイポグラフィ刷新:**
    *   Before: `Cinzel` (Display Serif) - 装飾的
    *   After: **`Noto Serif JP` (明朝体) + `Helvetica/Roboto` (San-serif)** - 知的で可読性が高い組み合わせに変更。

3.  **コンテンツ構成の最適化:**
    *   **Hero:** 英語ブランド名ではなく、**「日本語の機能的タグライン」**を中央に据える。
    *   **Problem:** 「乾燥」だけでなく「年齢による肌変化」も含める。
    *   **Science:** 成分やメカニズムを、感覚的な言葉だけでなく、より説得力のある見せ方（図解的なレイアウトなど）にする。

## 3. 実装タスクリスト (Implementation Tasks)

以下のステップで、コードレベルでの改修を実行します。

- [ ] **Task 1: デザインシステムの再定義 (Global Styles)**
    - アプリ全体の背景色をダークからライト（白・オフホワイト）に変更。
    - フォント設定を明朝体ベースの実直なスタイルに変更。
    - アクセントカラーを「水」「光」を想起させる色調に調整。

- [ ] **Task 2: Heroセクションの「オルビス化」**
    - 配置: 清潔感のある白背景に、モデル写真を大胆に配置（余白の美）。
    - コピー: 「Lumière Éclat」を控えめにし、「**科学する、素肌。**」のような日本語キャッチをメインにする。
    - デザイン: パララックスや過度な演出を抑え、静止画としての完成度（レイアウトの美しさ）を優先。

- [ ] **Task 3: Problem / BrandStory の統合と深化**
    - Problemセクションを「エイジングケアの気付き」というトーンに変更。
    - 単なる悩み解決ではなく、「なぜオルビスなのか（＝スキンケア発想）」への接続をスムーズにする。

- [ ] **Task 4: 「誠実さ」の実装 (Features & Science)**
    - テクスチャー画像や成分のアイコンなどを用い、視覚的に「肌への優しさ」と「効果」を証明するUIを追加。

この計画に基づき、まずは土台となるデザインシステムとHeroセクションの刷新から着手します。
